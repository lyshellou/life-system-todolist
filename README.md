# 人生目标待办系统

一个将**长期目标管理**和**日常待办**结合的个人生产力工具。围绕健康、工作、学习、人际、娱乐五个人生维度，帮你规划目标、追踪进度、养成习惯。

---

## 功能

| 功能 | 说明 |
|---|---|
| **五大维度** | 健康 / 工作 / 学习 / 人际 / 娱乐，每个维度独立管理目标和待办 |
| **长期目标** | 每个维度下可创建多个目标，附带里程碑节点和进度追踪 |
| **每日待办** | 支持独立待办和绑定到某个目标的待办 |
| **三桶视图** | 借鉴 Things 3 的「今天 / 随时 / 某天」三个桶管理待办优先级 |
| **经验等级** | 完成待办 +10 XP，里程碑 +25 XP，目标 +50 XP。积攒经验升级 |
| **连续打卡** | 每天完成待办自动记录连续天数，断了归零 |
| **成就徽章** | 10 个成就，条件达标自动解锁 |
| **活动热力图** | 最近 12 周每日活动量可视化 |
| **总览仪表盘** | 一眼看清各维度进度、经验值、今日任务 |

---

## 快速启动

本项目**不需要安装任何依赖**，只需 Node.js 即可运行。

```bash
# 在项目根目录执行
node server.js
```

在浏览器打开 **http://localhost:3001** 即可使用。

如果 `node` 不在默认路径，试试：
```bash
D:\360\node.exe server.js
```

## 操作指南

### 创建目标
1. 在总览页点击某个维度卡片进入详情
2. 点击「+ 新建目标」，填写标题和描述即可

### 创建待办
1. 在待办页点击「+ 新建待办」
2. 选择放入哪个桶（今天 / 随时 / 某天）、优先级、是否关联某个目标
3. 点击待办左侧方框标记完成

### 添加里程碑
在目标详情页点击「里程碑」栏的「+ 添加」，输入关键节点名称，完成后点击圆点勾选。

## 经验与等级

| 动作 | XP | 说明 |
|---|---|---|
| 完成待办 | +10 | 每次勾选待办即获得 |
| 里程碑达成 | +25 | 勾选里程碑时获得 |
| 目标完成 | +50 | 标记目标完成时获得 |

每升一级需要 `等级 × 100` XP：Lv1→Lv2 需要 100 XP，Lv2→Lv3 需要额外 200 XP，以此类推。

连续打卡期间每天完成待办额外加分（服务器自动处理）。

## 技术说明

- **后端：** Node.js 内置 `http` / `fs` / `path` 模块，零外部依赖
- **前端：** 原生 JavaScript SPA，无框架无打包器
- **数据：** 本地 `data/db.json` 文件存储
- **设计：** 基于 Minimalist 设计风格，Plus Jakarta Sans 字体
- **端口：** 3001
- **启动：** `node server.js` 即可运行

注意：这是一个本地个人工具，数据全部存储在 `data/db.json` 文件中。备份或迁移只需复制此文件即可。

如需对项目有更深入的技术了解，请阅读 `AGENTS.md` 文件（面向 AI Agent 的完整技术文档）。
